# Data visualisation course syllabus for context:

In this course, you'll dive into: 
- data visualization
- data storytelling

For this you will be using the following tools:
- Power BI
- Python
  - streamlit
  - pandas
  - matplotlib
  - duckdb

Prerequisites for this course are:
- Python fundamentals
- Pandas fundamentals

| **Week** | **Content**                                            | **Lectures** | **Exercise** |
| -------- | ------------------------------------------------------ | ------------ | ------------ |
| 1        | Intro, EDA, Power BI setup & semantic model            | 00-04        | 0, Project   |
| 2        | Power BI report/dashboard & design                     | 05, 07       | 1, Project   |
| 3        | Power BI publish & what if analysis, data storytelling | 08-10        | 2, Project   |
| 4        | Data storytelling (con't) & streamlit                  | 10, 11       | 3, Project   |
| 5        | -                                                      |              | Project      |
| 6        | -                                                      |              | Project      |
