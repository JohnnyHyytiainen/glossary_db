# Cloud Databricks and Azure course syllabus for context:

This course is divided into two major parts

1. Databricks part with focus on data engineering
2. Azure part with focus on cloud fundamentals

Prerequisites for this course are:

- python
- SQL
- pandas
- data visualizations

| 3        | dashboard, genie, deployment                                                         | 08-10        | lab          |
